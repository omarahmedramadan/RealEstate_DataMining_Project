# Real Estate Data Mining Project

**Team Members:** Omar Ahmed Ramadan & Ahmed Mohamed Fathy Aldmrdash

## 1. Project Overview

[cite_start]This project aims to conduct a comprehensive data mining study on the Real Estate domain using the **CRISP-DM** methodology[cite: 6]. [cite_start]The primary goal is to analyze property characteristics and build predictive models to understand the key factors influencing market prices and to provide accurate price forecasts[cite: 47].

## 2. Dataset & Preprocessing

[cite_start]We are using the **airbnb_cleaned.csv** dataset, which contains detailed property listings[cite: 23].
**Note:** The dataset has already undergone initial cleaning and preprocessing to ensure data quality, including:

* **Text Normalization:** Standardizing neighborhood and room type descriptions.
* [cite_start]**Handling Missing Values:** Addressing null entries to prevent model bias[cite: 47].
* **Outlier Detection:** Identifying and managing extreme price values using statistical methods.

## 3. Techniques to be Applied

[cite_start]To achieve our goals, we will implement the following data mining techniques[cite: 45]:

* **Regression:** To predict property prices based on features like location and room type.
* **Clustering:** To segment residential areas into distinct groups based on price patterns and availability.
* **Data Exploration:** Visualizing distributions and correlations between variables[cite: 47].
